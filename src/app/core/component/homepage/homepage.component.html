CDK & NGRX Workshop
You should be familiar with Angular Framework and RxJS
This workshop is spread across 4 weeks, having one technical presentation in each week.
  • Week 1 : CDK - Overlay
  • Week 2 : CDK - Portal
  • Week 3 : NG-RX - Actions & Store & Effects
  • Week 3 : NG-RX - Actions & Store & Effects
CDK
The Component Dev Kit (CDK) is a set of behavior primitives for building UI components.
Check official documentation here.
NG-RX
NgRx Store provides reactive state management for Angular apps inspired by Redux. Unify the events in your application and derive state using RxJS.
Check official documentation here.
